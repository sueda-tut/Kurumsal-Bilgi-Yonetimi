RAG (Retrieval-Augmented Generation): Cevap vermeden önce ilgili dokümanları bulup bunları kullanarak cevap üreten yapay zekâ yöntemidir.

Chunk: Büyük bir dokümanın yapay zekânın işleyebilmesi için bölündüğü küçük metin parçalarıdır.

Embedding: Bir metnin anlamını temsil eden sayısal vektöre dönüştürülmüş halidir.

Vector (Vektör): Embedding işlemi sonucunda oluşan sayılar dizisidir.

pgvector: PostgreSQL'de embedding vektörlerini saklayıp benzerlik araması yapmayı sağlayan eklentidir.

Semantic Search (Anlamsal Arama): Kelimeleri değil, metinlerin anlam benzerliğini kullanarak arama yapan yöntemdir.

OpenAI API: OpenAI modellerini uygulamamızdan kullanmamızı sağlayan programlama arayüzüdür.

API Key: OpenAI API'ye güvenli erişim sağlayan gizli kimlik anahtarıdır.

FastAPI: Python ile hızlı ve modern API geliştirmeyi sağlayan framework'tür.

SQLAlchemy: Python ile veritabanı işlemlerini SQL yazmadan yapmayı sağlayan ORM kütüphanesidir.

ORM (Object Relational Mapping): Veritabanı tablolarını Python sınıflarıyla eşleştiren yapıdır.

Pydantic: API'ye gelen ve giden verilerin doğruluğunu kontrol eden kütüphanedir.

Schema: API'de kullanılacak veri yapısını ve kurallarını tanımlayan şablondur.

CRUD: Veritabanında veri oluşturma, okuma, güncelleme ve silme işlemlerinin genel adıdır.

Router: API isteklerini ilgili fonksiyonlara yönlendiren katmandır.

Service Layer: Birden fazla işlemi belirli bir iş akışına göre yöneten katmandır.

JWT (JSON Web Token): Giriş yapan kullanıcıyı doğrulamak için kullanılan güvenli kimlik belirtecidir.

Authentication (Kimlik Doğrulama): Kullanıcının kim olduğunu doğrulama işlemidir.

Authorization (Yetkilendirme): Doğrulanan kullanıcının hangi işlemleri yapabileceğini belirleme işlemidir.

Token: Kimlik doğrulama veya yapay zekâ işlemlerinde kullanılan dijital veri birimidir.

Prompt: Yapay zekâya verilen komut veya talimattır.

Context (Bağlam): Yapay zekâya cevap üretmesi için verilen ilgili bilgi ve doküman parçalarıdır.

Hallucination (Halüsinasyon): Yapay zekânın gerçekte olmayan veya yanlış bilgiler üretmesidir.

Temperature: Yapay zekânın cevap üretirken ne kadar yaratıcı veya tutarlı olacağını belirleyen parametredir.

Token (Dil Modeli Token'ı): Yapay zekânın metni işlerken kullandığı ve ücretlendirmede esas alınan en küçük metin birimidir.
